# English Learning Game Hub

An interactive web application designed to help users learn and think in English through games and practical tools.

## 🌐 Live Demo
(Coming soon)

## 🚀 Features
- Flashcards for vocabulary learning
- Memory game (word matching)
- Listening quiz with audio (planned)
- Dictionary system to save and review words
- Custom deck creation (planned)

## 🧠 Concept

This project focuses on helping users think in English instead of translating, using interactive and engaging learning methods.
